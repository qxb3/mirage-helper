## Self hosting

### Installation

```bash
git clone https://github.com/qxb3/mirage-helper.git
cd mirage-helper
npm install #or yarn
```

> NOTE: Make a file called .env and look at .env.example
to see how to set it up

### Running the bot

```bash
npm run start #or yarn start
```

Simple as that :D
